# Introduction

Before we discuss the implementation of the Perceptron, let's take a look at **what a Perceptron is and why it is built**.

Warren McCulloch and Walter Pitts introduced the concept of a Perceptron in 1943. Basically it replicated the functioning of a simple neuron. It takes input signal and based on it's strength generate output signal. The Perceptron is the simplest form of an artificial neural network. It is a linear classifier, which means it can classify data that is linearly separable.

# Working of Perceptron

As you can see in the above image, the Perceptron takes $\mathbf{x1}$, $\mathbf{x2}$, $\mathbf{x3}$, ..., $\mathbf{xn}$ inputs and has $\mathbf{w1}$, $\mathbf{w2}$, $\mathbf{w3}$, ..., $\mathbf{wn}$ weights. For a better understanding of what inputs and weights are, let's consider a biological neuron. The inputs for a biological neuron are some information from the environment like *"Is it sunny?"* or *"Is it rainy?"* etc. Weights are the knowledge the neuron has to predict the output, like sunny weather can be observed by no clouds, no rain, bright light, etc. Neurons gain knowledge by learning from the environment.

You can easily predict the output because you have seen the weather and you know the weights. But the Perceptron doesn't know the weights initially. It learns the weights by training on the dataset. The Perceptron takes the inputs and weights and calculates the weighted sum of inputs and weights. If the weighted sum is greater than a threshold value, the Perceptron fires and gives the output as 1, otherwise 0.

Mathematically, weighted sum is calculated as:

$$
\mathbf{z} = \mathbf{w1} \times \mathbf{x1} + \mathbf{w2} \times \mathbf{x2} + ... + \mathbf{wn} \times \mathbf{xn}
$$

And, if $\mathbf{z} > \mathbf{threshold}$, then the output is 1, otherwise 0.

If the output is not correct, the Perceptron updates the weights using the Perceptron learning rule. The Perceptron learning rule is given by:

$$
\mathbf{w} = \mathbf{w} + \mathbf{η} \times (\mathbf{target} - \mathbf{output}) \times \mathbf{x}
$$


Where:
- $\mathbf{w}$ is the weight vector, $\mathbf{η}$ is the learning rate, $\mathbf{target}$ is the target output, $\mathbf{output}$ is the predicted output and $\mathbf{x}$ is the input vector

# Implementation

Let's start by creating a Perceptron class and initialize it.

```python title="Initialize Perceptron"
import numpy as np

class Perceptron(object):
    def __init__(self, lr=0.01, epochs=50):
        self.lr = lr
        self.epochs = epochs
```

In the above code, we have created a Perceptron class with two parameters, `lr` and `epochs`. `lr` is the learning rate and `epochs` is the number of iterations the Perceptron will train on the dataset.

Next, we will define the `fit` method, which will train the Perceptron on the dataset.

```python title="Fit method"
def fit(self, X, y):
    self.weights_ = np.random.normal(loc=0, scale=0.01, size=1+X.shape[1])
    self.errors_ = []

    for _ in range(self.epochs):
        errors = 0
        for xi, target in zip(X, y):
            update = self.lr * (target - self.predict(xi))
            self.weights_[1:] += update * xi
            self.weights_[0] += update
            errors += int(update != 0.0)
        self.errors_.append(errors)
    
    return self
```

In the `fit` method, we have initialized the weights using the normal distribution, which take the mean(`loc`), standard deviation(`scale`) as parameters and size which we take as `1+X.shape[1]`. We take one extra weight for the bias term. We have also initialized the `errors_` list to store the number of misclassifications in each epoch.
Then we iterate over the number of epochs and update the weights using the Perceptron learning rule. We calculate the update using the formula: 

$$
\mathbf{η} \times (\mathbf{target} - \mathbf{output}) \times \mathbf{x}
$$

We also update the bias term using the formula: 

$$
\mathbf{w0} += \mathbf{η} \times (\mathbf{target} - \mathbf{output})
$$

We calculate the number of misclassifications in each epoch and append it to the `errors_` list.

Next, we will define the `net_input` method, which calculates the weighted sum of inputs and weights.

```python title="Net input method"
def net_input(self, X):
    return np.dot(X, self.weights_[1:]) + self.weights_[0]
```

In the `net_input` method, we calculate the dot product of the input vector and weights and add the bias term to it, which is basically given by formula:

$$
\mathbf{z} = \mathbf{w1} \times \mathbf{x1} + \mathbf{w2} \times \mathbf{x2} + ... + \mathbf{wn} \times \mathbf{xn} + \mathbf{w0}
$$


Finally, we will define the `predict` method, which predicts the output based on the weighted sum.

```python title="Predict method"
def predict(self, X):
    return np.where(self.net_input(X) >= 0.0, 1, 0)
```

In the `predict` method, we calculate the weighted sum using the `net_input` method and return 1 if the weighted sum is greater than or equal to 0, otherwise 0.

So, this is the complete implementation of the Perceptron class. In the next section, we will train the Perceptron on the Iris dataset.

As Perceptron is a binary classifier, we can't classify these three classes at once. So, we will consider only two classes, Iris-setosa and Iris-versicolor. But we can extend the Perceptron to classify multiple classes using the One-vs-All(OvA) method. For now, let's train the Perceptron on Iris-setosa and Iris-versicolor.

Let's import the Iris dataset.

```python title="Import Iris dataset"
import pandas as pd

df = pd.read_csv('https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data', header=None)
df.head()
```

from the dataset, let's extract main features that can classify Iris-setosa and Iris-versicolor. As you can see in their images, the petal length and sepal length can be the main features to classify them. So here is the code to extract these features:

```python title="Extract main features"
import matplotlib.pyplot as plt
import numpy as np

#select setosa and versicolor
y = df.iloc[0:100, 4].values
y = np.where(y == 'Iris-setosa', 0, 1)

#extract sepal length and petal length
X = df.iloc[0:100, [0, 2]].values

#plot data
plt.scatter(X[:50, 0], X[:50, 1], color='red', marker='o', label='setosa')
plt.scatter(X[50:100, 0], X[50:100, 1], color='blue', marker='x', label='versicolor')

plt.xlabel('sepal length [cm]')
plt.ylabel('petal length [cm]')
plt.legend(loc='upper left')

plt.show()
```

In the above code, we have selected only the Setosa and Versicolor classes and extracted their sepal length and petal length. Setosa is represented by the red color and Versicolor by the blue color. For the target variable, Setosa is represented by 0 and Versicolor by 1.

```python title="Train Perceptron"
ppn = Perceptron(lr=0.1, epochs=10)
```
